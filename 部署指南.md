# 🚀 闲猫市集 · 上线部署指南

---

## 准备工作
- 一台电脑（Windows 或 Mac 都可以）
- 一个邮箱（用于注册账号）
- 整个过程约 20 分钟

---

## 第一步：注册 GitHub 账号（存放代码）

1. 打开浏览器，访问：https://github.com
2. 点击右上角 **"Sign up"**
3. 填写：
   - 邮箱地址
   - 密码
   - 用户名（如：xianmao-market）
4. 完成邮箱验证

---

## 第二步：创建仓库并上传代码

1. 登录 GitHub 后，点击右上角 **"+"** → **"New repository"**
2. 填写：
   - Repository name：`xianmao`
   - 选择 **Public**
3. 点击 **"Create repository"**
4. 在新页面点击 **"uploading an existing file"**
5. 把 `xianmao` 文件夹里**所有文件和文件夹**拖进去
   - ⚠️ 注意：要上传整个文件夹内容，包括 src 文件夹
6. 拉到底部点击 **"Commit changes"**

---

## 第三步：注册 Vercel 并部署

1. 打开：https://vercel.com
2. 点击 **"Sign Up"** → 选择 **"Continue with GitHub"**（用 GitHub 账号登录）
3. 授权后，点击 **"Add New Project"**
4. 找到你的 `xianmao` 仓库，点击 **"Import"**
5. 配置页面什么都不用改，直接点 **"Deploy"**
6. 等待 2-3 分钟，看到 🎉 就成功了！

---

## 第四步：获取你的网址

部署成功后，Vercel 会给你一个网址，格式是：

```
https://xianmao-XXXXXX.vercel.app
```

这就是你的 APP 网址，全球用户都可以访问！

---

## 第五步：绑定自定义域名（可选）

如果想要更专业的域名（如 xianmao.com）：

1. 去 https://www.namecheap.com 购买域名（约 RM40-60/年）
2. 在 Vercel 项目里点 **"Settings"** → **"Domains"**
3. 输入你的域名，按提示设置 DNS

---

## 管理后台使用方法

打开你的网址后：
1. 点击底部导航 **"我的"**
2. 滑到最底部，找到 **"管理员入口"**
3. 输入密码：**admin888**
4. 进入管理后台

### 后台功能：
| 功能 | 说明 |
|------|------|
| 📊 数据总览 | 查看商品数、用户数、待处理事项 |
| 📦 商品管理 | 审核/上架/下架/删除商品 |
| 👥 用户管理 | 封禁/解封用户，手动认证 |
| 🚩 举报处理 | 处理用户举报 |
| ⚙️ 平台设置 | 修改平台名称、公告、手续费等 |
